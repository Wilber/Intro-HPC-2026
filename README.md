
# Intro to HPC Bootcamp 2026  
Repository for training materials, project resources, and pre‑requisite content for the **Intro to HPC Bootcamp**, held in person at **Argonne National Laboratory (ANL)** in **August 2026**. The Bootcamp is hosted by the **Argonne Leadership Computing Facility (ALCF)**.

---

## Overview  
This repository contains all materials supporting the *Intro to HPC Bootcamp 2026*, including:

- Project-based exercises  
- Pre‑requisite reference materials  
- Instructor notes  
- Example scripts  
- Links to external documentation and HPC resources  

Use this repository as a working area to experiment with HPC concepts, examples, and hands‑on coding exercises.

---

## Repository Structure  

**Project1–Project10**  
Each folder contains workshop materials, example code, exercises, and guidance for a specific bootcamp project.

**Pre-req-resources**  
Contains materials on foundational HPC concepts, Linux basics, introductory parallel programming, ALCF account setup, SSH guidance, and suggested background readings.

---

## Goals of the Bootcamp  
- Introduce high‑performance computing concepts  
- Teach users how to run and optimize jobs on large-scale systems  
- Explore parallel programming models (MPI, OpenMP, GPU computing, etc.)  
- Provide hands-on experience using ALCF resources  
- Develop practical workflows used in scientific computing

## Contributing  
This repository is collaborative. All colleagues contributing materials should follow the workflow below.

### Creating a Branch  
```
git checkout -b <branch-name>
```

Example:  
```
git checkout -b add-project3-examples
```

### Making Changes  
```
git add .
git commit -m "Describe your change clearly"
```

### Pushing Your Branch  
```
git push -u origin <branch-name>
```

### Opening a Pull Request  
After pushing your branch, go to:
https://github.com/Wilber/Intro-HPC-2026/pulls

- Select “New Pull Request”  
- Choose your branch  
- Describe your changes  
- Submit for review
